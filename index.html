Personal Portfolio Page

My goal was to create unique visuals presenting important information about how can I help you with my skills, and why I do what I do. I kinda like the elegance in stuff, so I decided to use mostly light text on dark background with a bit of "classy touch" (yeah... that royal purple, heh).

The site is static, but can forward messages. Sections are created using custom Bootstrap layout. Most of the visual formating is created with pure CSS, part of the elements are powered by javaScript libraries and custom javaScript.

The result is a website that looks great on any device.
